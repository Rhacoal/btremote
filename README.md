# btremote
## 简介
基于传统蓝牙的小车遥控程序，提供了连接到已配对设备和自动重连的功能。

### 测试环境
在以下环境下正常编译运行：
- Windows 10 1909 Qt 5.12.4
- Mac OS X 10.15.5 Qt 5.12.1
